# external_library_docs.md

When trying to learn about a library we're using, do **NOT** start by searching the web for the docs.

## Guidelines

- In our repo find the directory `docs` inside of `.reference` at [`references`](../../.reference/docs/). There we have a lite of directories.
- Inside of each directory we have a `README.md file for the library
- Sometimes we also have a **big** directory with the entire documentation in `.md` or in `.rst`.
- Feel free to use those to guide you on how to use an specific library.
- The libraries `returns`, `classes` and `lambdas` have code as well since we'll build a lot of things on top of it.
